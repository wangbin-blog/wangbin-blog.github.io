# 激活

Typora 手动激活，不用修改时间，不用补丁

下面是操作步骤：

  1. 下载最新版本的 Typora 安装

  2. 安装完后，进入 typora 的安装目录下的 \resources\page-dist\static\js 目录，找到 LicenseIndex 开头的文件

  3. 用文本编辑器打开该文件，搜索 hasActivated="true"==e.hasActivated 并将其替换为
    hasActivated="true"=="true"

    [!IMPORTANT]

    （修改前，可将该文件备份一下，万一操作失误，可以进行恢复）

  4. 打开软件尽情使用吧！
