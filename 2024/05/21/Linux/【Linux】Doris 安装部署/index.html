# 4. йғЁзҪІ fe

  • дҝ®ж”№ FE й…ҚзҪ®ж–Үд»¶

    StarRocks еҗ„жңҚеҠЎй…ҚзҪ®ж–Үд»¶зҡ„й»ҳи®Өй…ҚзҪ®е·Із»Ҹи¶ід»ҘеҗҜеҠЁйӣҶзҫӨпјҢдёҚе»әи®®еҲқе°қз”ЁжҲ·дҝ®ж”№иҫғеӨҡй…ҚзҪ®гҖӮFE зҡ„й…ҚзҪ®ж–Үд»¶еңЁжөӢиҜ•зҺҜеўғдёӯжҲ‘д»¬йҖҡеёёеҸӘйңҖиҰҒе…іжіЁд»ҘдёӢдёүзӮ№пјҡ

    aгҖҒжіЁж„Ҹй»ҳи®Өз«ҜеҸЈпјҢйҒҝе…Қз«ҜеҸЈеҶІзӘҒпјҢжӯЈеёёжғ…еҶөдёӢдёҚйңҖиҰҒдҝ®ж”№пјӣ

    bгҖҒз»‘е®ҡ IPпјҢйҒҝе…ҚеӨҡзҪ‘еҚЎжғ…еҶөдёӢжңҚеҠЎж— жі•иҮӘеҠЁеҢ№й…ҚеҲ°жӯЈзЎ®зҡ„ IPгҖӮзү№еҲ«жіЁж„ҸпјҢз»‘е®ҡ ip ж—¶пјҢеҰӮжһңдёҚжё…жҘҡ CIDR иЎЁзӨәжі•пјҢеҸҜд»ҘзӣҙжҺҘеЎ«еҶҷе®Ңж•ҙзҡ„ ipпјҲдёҚж”ҜжҢҒеҲ«еҗҚпјүпјҢдҫӢеҰӮй…ҚзҪ®дёә priority_networks = 172.18.88.54/24; иҜҘең°еқҖеҸҜд»ҘдҪҝз”Ё ip addr show зҡ„ж–№ејҸжҹҘиҜў
