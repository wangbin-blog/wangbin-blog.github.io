# 安装

  1. 安装好 docker 后，运行如下命令创建 bridge 网络： docker network create maccmsnet

    • 查询到新创建的 bridge maccmsnet

    • 查看网桥下的容器 ip 信息： docker network inspect maccmsnet

  1. 运行容器连接到 maccmsnet 网络

    • Mysql 安装

      • --name 容器名称
      • -e 配置时间区域
      • -p 主机端口和容器端口映射
      • --restart 自动重启
      • -v 配置文件 / 数据文件映射主机
      • --network 网络配置
      • --network-alias 网络别名

    • Nginx 安装

    • PHP 安装

  2. 上传 maccmsV10 项目文件

下载地址： https://github.com/magicblack/maccms10/releases

下载完成后解压文件，并做一些处理

  1. 配置 Nginx

    程序文件准备好了，而且放在了 /opt/docker/nginx/webs 目录，nginx 和 php 都可以访问到，接下来修改一下 Nginx 的配置文件

  2. 根据向导安装，安装完成后修改 admin.php 文件名，同时修改 nginx 配置

  3. 配置视频分类

  4. 最后就可以开始采集了
