JadeAI：AI 驱动的智能简历构建器，让你的简历脱颖而出

Think.Wang2026-03-242026-03-26

嗨，职场人们！

找工作时，一份专业、吸引人的简历是打开职场大门的第一把钥匙。但制作一份优秀的简历往往需要花费大量时间和精力，尤其是在面对不同岗位的 JD 时，需要不断调整和优化。

今天要给你们推荐一个超级好用的工具——JadeAI，一个 AI 驱动的智能简历构建器，让你的简历制作变得简单高效！

什么是 JadeAI？

JadeAI 是一个开源、免费的 AI 智能简历构建器，它提供了丰富的功能，帮助你轻松创建专业、个性化的简历。无论是刚毕业的学生还是职场老手，都能通过 JadeAI 制作出令人印象深刻的简历。

核心功能介绍

1️⃣ 直观的简历编辑

拖拽式编辑器：可视化排列和重新排序简历部分和项目，操作简单直观。

内联编辑：直接在画布上点击任何字段进行编辑，所见即所得。

50+ 专业模板：提供经典、现代、极简、创意、ATS 友好、时间线、北欧、瑞士等多种风格的模板。

主题定制：可自定义颜色、字体、间距和边距，实时预览效果。

编辑历史：支持最多 50 步的撤销/重做操作，不用担心操作失误。

自动保存：可配置保存间隔（0.3s-5s），也支持手动保存。

Markdown 支持：在文本字段中使用 Markdown 语法格式化内容，如 粗体、代码、项目符号列表等。

2️⃣ 强大的 AI 功能

AI 聊天助手：编辑器中集成的会话式 AI，支持多会话和持久历史记录。

AI 简历生成：根据职位、经验和技能生成完整的简历。

简历解析：上传现有的 PDF 或图像，AI 自动提取所有内容。

JD 匹配分析：将简历与职位描述进行比较，提供关键词匹配、ATS 评分和改进建议。

求职信生成：基于简历和 JD 生成 AI 定制的求职信，可选择语气（正式/友好/自信）。

语法和写作检查：检测弱动词、模糊描述和语法问题，返回质量评分。

翻译：在保留技术术语的同时，将简历内容翻译成 10 种语言。

灵活的 AI 提供商：支持 OpenAI、Anthropic 和自定义 API 端点，每个用户在应用内配置自己的密钥。

3️⃣ 多格式导出与分享

多格式导出：PDF（Puppeteer + Chromium）、智能单页 PDF（自动适应单页）、DOCX、HTML、TXT、JSON。

JSON 导入：导入之前导出的 JSON 文件以恢复或创建简历，支持在编辑器中（覆盖当前）和仪表板上（创建新）。

链接分享：基于令牌的可共享链接，可选密码保护。

查看计数器：跟踪共享简历被查看的次数。

4️⃣ 简历管理

多简历仪表板：网格和列表视图，搜索，排序（按日期、名称）。

从 JSON 导入：直接在仪表板上从 JSON 导出创建新简历。

复制和重命名：快速简历管理操作。

交互式导览：为首次用户提供分步入门指南。

5️⃣ 其他功能

双语界面：完整的中文（zh）和英文（en）界面。

暗黑模式：支持浅色、深色和系统主题。

灵活的认证：Google OAuth 或浏览器指纹（零配置）。

双数据库：SQLite（默认，零配置）或 PostgreSQL。

快速开始

Docker 部署（推荐）

  1. 生成密钥

  2. 运行容器

  3. 访问应用
    打开 http://localhost:3000。数据库在首次启动时会自动迁移和种子。

    注意：AUTH_SECRET 是会话加密所必需的。使用 openssl rand -base64 32 生成一个。

AI 配置

无需服务器端 AI 环境变量。每个用户在应用内的设置 > AI 中配置自己的 API 密钥、基础 URL 和模型。

使用 PostgreSQL

应用场景

JadeAI 特别适合以下场景：

  • 求职者：快速创建和优化简历，提高求职成功率
  • 职场人士：更新简历，适应不同岗位的需求
  • 招聘人员：分析简历与职位的匹配度
  • 学生：制作第一份专业简历，迈出职场第一步

为什么选择 JadeAI？

  1. 开源免费：完全开源，免费使用，无隐藏费用
  2. AI 驱动：利用 AI 技术优化简历，提高竞争力
  3. 功能丰富：提供从编辑到导出的完整功能
  4. 易于部署：一键 Docker 部署，快速开始使用
  5. 高度定制：支持多种模板和主题，满足个性化需求
  6. 多语言支持：支持中英文界面和多语言翻译

总结

JadeAI 是一个功能强大、易于使用的 AI 智能简历构建器，它将 AI 技术与简历制作相结合，帮助用户创建专业、有针对性的简历。无论是寻找第一份工作还是寻求职业发展，JadeAI 都能为你提供有力的支持。

如果你正在为简历制作而烦恼，不妨试试 JadeAI，让 AI 帮助你打造一份脱颖而出的简历！
