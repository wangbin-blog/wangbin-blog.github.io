【漫画/小说/影视/音乐】益达APP

发表于 2024-10-29 更新于 2024-11-26 分类于 工具 ， 娱乐 阅读次数： 本文字数： 101 阅读时长 ≈ 1 分钟
“益达APP”，一个基于 Flutter + Rust 开发的跨平台娱乐应用。该应用支持小说、影视、漫画和音频等多种内容类型，适用于安卓、iOS、Mac 和 Windows 平台。文章提供了安装包下载链接、规则导入链接和规则说明文档，帮助用户更好地使用该应用。此外，还提到了第三方源地址，方便用户扩展内容资源。

简介

平台：

  • 全平台(安卓\IOS\MAC\WIN)

类型：

  • 小说
  • 影视
  • 漫画
  • 音频

flutter + rust 开发! 跨平台

  • 安装包获取:https://github.com/xiaohucode/yidaRule/releases
  • 规则导入：https://github.com/xiaohucode/yidaRule/raw/main/siteRule/ydSource.yds
  • 规则说明文档: https://github.com/xiaohucode/yidaRule/blob/main/Document.md
  • PS: 规则中有很多失效的了，重在学习怎么制作源！
  • 源第三方地址：https://gcore.jsdelivr.net/gh/xiaohucode/yidaRule/siteRule/ydSource.yds
